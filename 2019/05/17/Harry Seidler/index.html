Harry Seidler

发表于 2019-05-17 | 分类于 Architecture Notes |

—悉尼充满了哈里赛德勒的设计，每天都可以看到家门口的索普游泳馆（每次看着都觉得是人名，直到念出来才意识到是索普）；去办美签的时候感受到了优秀服务的MLC电梯系统；刚到悉尼的第一个周五晚上误入到Australia Square的Open Plaza，摇滚音乐节也就这么多人了，穿着正装在CBD工作的男男女女，不大声喊是听不到对方说话的；学Revit时候建的模型Rose Seidler House;以及在悉尼看见栋高楼查一下就发现是他设计的（但是一旦查过建筑年份之后会觉得，即便是几十年前的设计，现在确依然不会觉得过时）。

简介

–作为澳洲最出名的建筑师之一，搜索名字作为关键字会有很多介绍。百科介绍可以涵盖到每个方面（Harry Seidler的夫人竟然是Evatt家族的人，澳洲社会好小的感觉~ PS维基百科真的感觉很好用），建筑特点“social use，efficient building construction methods and visual aesthetics（还是这三个基本点，但是会根据大环境的变化，新技术的开发而与时俱进）

–知网的这篇文章（可以免费下载），可以大概入门级了解到风格定位。（但是香港汇丰银行是福斯特设计的）

作品
  • http://seidler.net.au/是事务所的网站，会有之前哈里赛德勒的设计，也会有近几年的设计，需根据下方介绍。
  • http://www.canberrahouse.com.au/people/harry-seidler.html一个相对完整的居住建筑列表
