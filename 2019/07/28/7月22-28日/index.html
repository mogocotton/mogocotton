7月22-28日--WEEK 3

发表于 2019-07-28 | 分类于 44 Days in U.S |

– 第三周，节奏上来了啊宝贝们~

  • 7月22日： pin-up完美展示了我听老师话的操作~秉持着每天都没有虚度的课程设置，在pin-up后还会进行lecture的课程，好吧，来到了我很蒙蔽的figure ground课程，也是成功把底子变的面目全非的过程。

  • 7月23日： Morgan展示完课程所学之后，下午的炫技时间，忘了谁问他的作品集，然后被他含糊的过去了，后来被Hannah成功的在网上找到了，哈哈哈哈哈，我们又成功的秀给了我一直以为是越南小哥的哥看（都是被他family name莫名其妙的拼写骗了，人家是华裔不说，还会儿说点儿普通话，而且最后被我培养成了只跟我讲中文的习惯）

  • 7月24日：周三下午的固定组合校外人员（会以自身的经历结合来扩展讲他们做过的项目，给大家一个概念，之后你会做些什么，让你自己来判断你是否会对这个行业感兴趣。因为很多学生是UCLA或者UC系统自己的学生，他们大三才有建筑专业出现，如果想转专业或转学，提前来修下学分，且不管什么样，有东西就都给A还是很难得的，是与我们国家完全不同的教育体系设置，不过我们近几年也出现了这种先上大学并不分专业的改革，我们的高中课程设置并不能让我知道我们到底对哪方面感兴趣，时代总会变的~）来做一个讲座加逛校园，通常都会早下课，对于课程设置了有了清醒认识后（因为听前一年的人说，后两周只会更紧张，会有通宵的出现），抓紧碎片时间去了好莱坞，再次更新几个印象，以为星光大道，是一条华丽的大道，实际只是前面的一条路给你按上颗星星，竟然看到了川普的星星，他果然是娱乐明星来的；中国剧院为什么长得那么的牛鬼蛇神的样子（科普设计者）。

  • 7月25日：忘了干了什么，只记得自己不知道从什么时候开始，晚上开始留下画图了，鉴于从到澳洲后就比较放松的状态，突然被美国抓了一下态度，很多年没有上学不说，也找一下以前工作的感觉~就是以为是划水课程，结果每个人都很认真，我整个月都在尝试前五到studio，正常情况下10点上课，我一般9点左右就会到，可是每次挤不进前五，我只有在8点半到的那次勉强得到了第五名，大家都太拼了（我是最没压力的，不需要学分，不需要任何经历证明，但是很莫名的跟着他们下来了）

  • 7月26日： 展示一下自己挑选的空间，讲述一下它怎么个有趣法~模型的制作方法课程也更上了，学如何操作机器，看别人的模型~为自己的做前期准备，每个人会有5lb的rockite由学校提供，自己再买点儿~

  • 7月27日： 因为是和同学在澳洲就约好的行程，只能卖力的搬着电脑，倒出租公交到Laguna Beach看一下这个神奇的人物表演，望远镜必须，是大家画好超级平面的妆，然后扮演画里的人，只有偶尔动一下才会让你意识到这是个真人而不是画，完整的做完了长滩岛到圣克莱门第的公交车全程（因为夏天到了，流浪汉要到车上纳凉，车上总是会有奇怪的味道，流浪汉穿衣不分四季不知道为什么~）；因为下周要开始做模型了，要自己浇筑混凝土了，要提前准备rockite，到现在都不知道对应的中文是什么，老外周末不送快递真的是十分不方便了，想到了国内快递业务的好（所以在国外被很多人问过，尤其在美国问的人很多，是留在国外还是回国，当然是回国了，国内生活超方便的~）

  • 7月28日： 虽然被网站骗了，不能成功提货，但是幸亏西好莱坞的那家店有两桶，成功的拿到了我和Kathy的Rockite，最先定的两桶也最后成功的被别人用上了（拎回学校还给放到Max那儿了，哈哈哈哈哈，桌子找错了，还被Uber司机说没人在学校，只有你还在~）去买画材，进了blick就不想出来，看到了好多calligraphy的笔，比澳洲多且便宜，就要凑齐不同宽度的平行笔了的感觉（国外和国内卖东西的方式不同，会有大体量的分门别类的商店，卖画材就是很大的仓储卖画材可以看实物，卖五金澳洲会有bunnings这种，之前在泰国的时候就会觉得这种淘宝线下店对于挑东西真是太方便了~国内线下实体店会比较少，都在淘宝解决了一切）经此夏校我已经在各种交通工具上都捧着电脑画过图了。
